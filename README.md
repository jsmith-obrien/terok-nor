### What is Terok Nor?

**Terok Nor** was a Cardassian mining facility. It also is a locally hosted, offline-capable chatbot server designed for real work and real weirdness. It runs entirely on your own machine, with no cloud calls, no telemetry, and no judgment. Powered by open-source LLMs like LLaMA via Ollama, it serves a chat interface at `localhost:7860` where you can brainstorm, vent, write, edit, or just talk to something smarter than Clippy.

Use it for resume rewrites, therapy-lite sessions, automation ideas, or existential tech banter. Your data stays local. Your thoughts stay yours. Your assistant gets better with every install.
