# 🖖 Terok Nor – Text Generation Web UI

A local, customizable web interface for running large language models (LLMs) like LLaMA, Mistral, and others. This repo is based on the excellent `text-generation-webui`, customized for local experimentation.

## 🚀 Quickstart

```bash
git clone https://github.com/jsmith-obrien/terok-nor.git
cd terok-nor
bash start_linux.sh
